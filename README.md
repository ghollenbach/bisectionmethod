# Autointern Website
This is built using react.js


Here is all the code for the website side of things. It references the api built in the autointernapi repo. The App.jsx file is used for directing everything. Banner and Home are both used on the initial page. 

## Use
Use.jsx and its accompanying css file are used for detail submission for letter writing. It takes an upload of a resume as well as several fields

## Search
Calls the mentor search methods of the autointernapi repo.
